# 语音日历 (Voice Calendar)

通过语音交互管理日程安排的 Web 工具。

## 项目背景

传统日历需要手动输入事件，操作繁琐。本项目通过语音交互，让用户能够更自然、快捷地管理日程安排。

## 目标用户

需要快速记录和查看日程的普通用户，尤其是在双手不方便操作时（如开车、做饭）。

## 核心功能

| 功能       | 描述                                  | 优先级 |
| ---------- | ------------------------------------- | ------ |
| 语音添加事件 | 说出"X月X日X点做某事"，自动解析并存入日历 | 高     |
| 语音查看事件 | 说出"我X天有什么安排"，语音播报结果      | 高     |
| 语音删除事件 | 说出"删除X事件"，从日历中移除            | 高     |
| 日历界面展示 | 以月视图展示所有已添加事件               | 高     |
| 事件列表视图 | 列表形式展示近期所有事件                 | 中     |
| 语音播报反馈 | 每次操作后语音确认结果（如"已添加成功"）   | 中     |

## 范围边界（V1 不做）

- 用户登录/注册
- 多设备云同步
- 手机原生 App
- 日历导入/导出（iCal 格式）
- 重复事件（如每周一固定事项）
- 事件编辑

## 技术方案

| 模块           | 技术                           | 原因                       |
| -------------- | ------------------------------ | -------------------------- |
| 语音识别（输入） | Web Speech API                 | 浏览器自带，无需申请，免费    |
| 语音播报（输出） | Web Speech Synthesis API       | 浏览器自带，无需申请，免费    |
| 数据存储        | localStorage                   | 无需后端，简单可靠           |
| 界面框架        | 原生 HTML/CSS/JavaScript       | 无需额外框架                |
| 时间解析        | 自然语言解析（Claude API）       | 准确理解"明天""下周一"等模糊表达 |

## 语音交互示例

| 用户说               | 系统动作                       |
| -------------------- | ------------------------------ |
| "明天下午三点开会"     | 添加事件：明天 15:00 开会        |
| "我明天有什么安排"     | 语音播报明天所有事件              |
| "删除明天的开会"       | 删除对应事件，语音确认            |
| "这周五有什么事"       | 语音播报周五所有事件              |

## 本地运行

1. 克隆仓库
2. 在 `parser.js` 中填入你的 Claude API Key
3. 用 Chrome 浏览器打开 `index.html`（需麦克风权限，需联网）

## 已知限制

- Web Speech API 需要 Chrome 浏览器，且需要麦克风权限
- 语音识别依赖网络连接
- 数据仅存储在本地浏览器，清除缓存会丢失
